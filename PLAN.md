# برنامه بازی حافظه کودکان (Kids Memory Game)

## نمای کلی پروژه
بازی حافظه‌ای برای کودکان با استفاده از موتور Ursina که در آن کارت‌هایی با اعداد روی آن‌ها وجود دارد و کودکان باید جفت‌های یکسان را پیدا کنند.

---

## فاز اول: پایه‌گذاری (Foundation)

### ۱. راه‌اندازی موتور Ursina
- تنظیمات پنجره (عنوان، اندازه، رنگ پس‌زمینه)
- تنظیمات دوربین (موقعیت، نوع پروجکشن)
- مقداردهی اولیه محیط بازی

### ۲. کلاس NumberCard
کارت‌هایی که قابلیت نمایش اعداد دارند:
- **ویژگی‌ها:**
  - موقعیت روی صفحه (x, y)
  - عدد روی کارت
  - وضعیت (رو شده / پشت)
  - رنگ و استایل
  
- **قابلیت‌ها:**
  - انیمیشن چرخش نرم (Flip Animation) هنگام کلیک
  - نمایش عدد پشت کارت پس از چرخش
  - مدیریت رویداد کلیک
  
### ۳. کلاس GameManager اولیه
مدیر بازی که کارت‌ها را مدیریت می‌کند:
- **مسئولیت‌ها:**
  - ایجاد و چیدمان کارت‌ها روی صفحه
  - تعیین موقعیت هر کارت به صورت Grid
  - امکان تغییر تعداد کارت‌ها به صورت داینامیک
  
---

## فاز دوم: منطق بازی (Game Logic)

### ۱. سیستم Match Making
- تشخیص جفت‌های یکسان
- نگهداری کارت‌های باز شده
- پاک کردن یا پنهان کردن کارت‌های جفت شده

### ۲. مدیریت نوبت
- محدود کردن تعداد کارت‌های قابل باز شدن (دو کارت در هر نوبت)
- زمان نمایش کارت‌های نامطابق قبل از پوشیده شدن مجدد

### ۳. شمارش امتیازات
- سیستم امتیازدهی
- شمارش تلاش‌ها
- زمان‌سنج

---

## فاز سوم: بهبودها و جلوه‌ها (Enhancements)

### ۱. جلوه‌های بصری
- انیمیشن‌های نرم‌تر
- افکت‌های ذره‌ای (Particle Effects)
- رنگ‌های جذاب‌تر

### ۲. صداگذاری
- صدای چرخش کارت
- صدای جفت شدن صحیح
- صدای پس‌زمینه

### ۳. سطوح مختلف سختی
- آسان: 4x3 کارت (6 جفت)
- متوسط: 4x4 کارت (8 جفت)
- سخت: 6x4 کارت (12 جفت)

## نکات فنی

### تکنولوژی‌ها:
- **Python 3.8+**
- **Ursina Engine** - برای گرافیک و موتور بازی
- **پیش‌نیازها:** `pip install ursina`

### ساختار پروژه:
```
kids-memory-game/
├── main.py           # فایل اصلی اجرا

├── README.md        # مستندات
└── requirements.txt # وابستگی‌ها
```

### نکات برنامه‌نویسی:
- از OOP برای ساختار پروژه استفاده کنید
- کد را تمیز و قابل خواندن نگه دارید

- منطق را به صورت Modular و قابل توسعه بنویسید
