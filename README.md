# بازی حافظه کودکان (Kids Memory Game)

یک بازی حافظه‌ای جذاب برای کودکان که با استفاده از موتور Ursina ساخته شده است.

## نصب و راه‌اندازی

### پیش‌نیازها
- Python 3.8 یا بالاتر

### نصب وابستگی‌ها
```bash
pip install -r requirements.txt
```

### اجرای بازی
```bash
python main.py
```

## فاز اول (تکمیل شده) ✓

فاز اول شامل موارد زیر است:

1. **راه‌اندازی موتور Ursina**
   - تنظیمات پنجره (عنوان، اندازه، رنگ)
   - تنظیمات دوربین (orthographic projection)

2. **کلاس NumberCard**
   - نمایش کارت‌ها با رنگ و استایل مناسب
   - انیمیشن چرخش نرم هنگام کلیک
   - نمایش عدد پس از چرخش
   - علامت سوال روی پشت کارت

3. **کلاس GameManager**
   - ایجاد کارت‌ها به صورت داینامیک
   - چیدمان کارت‌ها در یک Grid منظم
   - قابلیت تغییر تعداد جفت‌ها

## فاز دوم (تکمیل شده) ✓

فاز دوم منطق اصلی بازی را پیاده‌سازی می‌کند:

1. **سیستم تشخیص تطبیق**
   - بررسی خودکار دو کارت باز شده
   - تشخیص جفت‌های یکسان
   - انیمیشن و امتیازدهی برای جفت‌های موفق

2. **مدیریت نوبت (1-5 بازیکن)**
   - پشتیبانی از 1 تا 5 بازیکن
   - نمایش بازیکن فعلی
   - چرخش خودکار نوبت

3. **قفل کلیک**
   - جلوگیری از کلیک روی کارت سوم
   - حفاظت در برابر کلیک‌های سریع

4. **تأخیر یادگیری**
   - 1.5 ثانیه تأخیر برای کارت‌های نامطابق
   - فرصت برای به خاطر سپردن موقعیت کارت‌ها

5. **سیستم امتیازدهی**
   - ثبت امتیاز هر بازیکن
   - تشخیص برنده در پایان بازی
   - پشتیبانی از حالت مساوی

6. **محدوده سطح**
   - قابلیت تنظیم محدوده اعداد (1-10، 11-20، و غیره)
   - مناسب برای یادگیری مرحله‌ای

7. **بازخورد واضح**
   - پیام‌های بصری برای موفقیت/شکست
   - نشانگرهای رنگی برای بازیکن فعلی

## فاز سوم و چهارم (تکمیل شده) ✓

فاز سوم و چهارم جلوه‌های بصری و صوتی را اضافه می‌کنند:

1. **سیستم Texture (تصاویر)**
   - پشتیبانی از تصاویر PNG از `assets/textures/`
   - Fallback خودکار به متن اگر تصویر نباشد
   - فرمت: `{number}.png` (مثلاً `1.png`, `2.png`)

2. **انیمیشن‌های نرم‌تر**
   - انیمیشن پرش (bounce) با `curve.out_back`
   - سه مرحله: پرش → برگشت → محو
   - جذاب‌تر و تأثیرگذارتر

3. **صداگذاری (Audio)**
   - صدای کلیک: `click.wav` هنگام کلیک روی کارت
   - صدای موفقیت: `success.mp3` برای جفت درست
   - صدای اشتباه: `wrong.wav` برای جفت نادرست
   - مدیریت خودکار در صورت نبود فایل‌های صوتی

4. **سیستم ذرات (Particles)**
   - ذرات رنگی confetti هنگام موفقیت
   - 8-12 ذره با رنگ‌های متنوع
   - فیزیک واقعی با گرانش
   - چرخش و محو شدن تدریجی

5. **بهبود خوانایی اعداد**
   - افزایش اندازه متن (scale 4)
   - رنگ واضح‌تر (سیاه روی سفید)
   - تمرکز بر یادگیری دانش‌آموزان

## نحوه بازی

### حالت تک‌نفره (اعداد 1-10)
```python
GameManager(num_pairs=10, num_players=1, level_start=1)
```

### حالت دو نفره (اعداد 11-20)
```python
GameManager(num_pairs=10, num_players=2, level_start=11)
```

### حالت پنج نفره (اعداد 1-6)
```python
GameManager(num_pairs=6, num_players=5, level_start=1)
```

## دارایی‌های بازی (Assets)

### تصاویر (assets/textures/)
برای استفاده از تصاویر به جای متن:
- تصاویر PNG با فرمت `{number}.png`
- اندازه توصیه‌شده: 256×256 پیکسل
- پس‌زمینه شفاف
- عددهای واضح و رنگی

### صداها (assets/sounds/)
برای جلوه‌های صوتی:
- `click.wav` - صدای کلیک کارت
- `success.mp3` - صدای موفقیت
- `wrong.wav` - صدای اشتباه

**توجه:** بازی بدون دارایی‌ها هم کار می‌کند (با fallback به متن و بدون صدا)
