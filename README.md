# بازی حافظه کودکان (Kids Memory Game)

یک بازی حافظه‌ای جذاب برای کودکان که با استفاده از موتور Ursina ساخته شده است.

## نصب و راه‌اندازی

### پیش‌نیازها
- Python 3.8 یا بالاتر

### نصب وابستگی‌ها
```bash
pip install -r requirements.txt
```

### اجرای بازی
```bash
python main.py
```

## فاز اول (تکمیل شده) ✓

فاز اول شامل موارد زیر است:

1. **راه‌اندازی موتور Ursina**
   - تنظیمات پنجره (عنوان، اندازه، رنگ)
   - تنظیمات دوربین (orthographic projection)

2. **کلاس NumberCard**
   - نمایش کارت‌ها با رنگ و استایل مناسب
   - انیمیشن چرخش نرم هنگام کلیک
   - نمایش عدد پس از چرخش
   - علامت سوال روی پشت کارت

3. **کلاس GameManager**
   - ایجاد کارت‌ها به صورت داینامیک
   - چیدمان کارت‌ها در یک Grid منظم
   - قابلیت تغییر تعداد جفت‌ها

## نحوه بازی
- با کلیک روی هر کارت، عدد پشت آن نمایش داده می‌شود
- کارت‌ها با انیمیشن نرم چرخش می‌کنند
