# Assets Directory

این پوشه شامل دارایی‌های بصری و صوتی بازی است.

## textures/
تصاویر PNG برای نمایش اعداد روی کارت‌ها

فرمت نام‌گذاری: `{number}.png`
مثال:
- `1.png` - تصویر عدد 1
- `2.png` - تصویر عدد 2
- `10.png` - تصویر عدد 10

اگر تصویری برای یک عدد موجود نباشد، بازی به صورت خودکار از متن استفاده می‌کند.

### راهنمای ایجاد تصاویر:
- اندازه توصیه شده: 256x256 پیکسل
- فرمت: PNG با پس‌زمینه شفاف
- استایل: عددهای واضح، رنگی، و کودک‌پسند
- فونت: بزرگ و خوانا برای کودکان کلاس سوم

## sounds/
فایل‌های صوتی برای جلوه‌های بازی

فایل‌های مورد نیاز:
- `click.wav` - صدای کلیک روی کارت
- `success.mp3` - صدای موفقیت در پیدا کردن جفت
- `wrong.wav` - صدای اشتباه در انتخاب جفت

### راهنمای صداها:
- صدای کلیک: کوتاه و ملایم (< 0.5 ثانیه)
- صدای موفقیت: شاد و انگیزه‌بخش (1-2 ثانیه)
- صدای اشتباه: ملایم و غیر منفی (< 1 ثانیه)

## توجه
اگر فایل‌های صوتی یا تصویری موجود نباشند، بازی به صورت خودکار:
- از متن برای نمایش اعداد استفاده می‌کند
- بدون صدا اجرا می‌شود

این رویکرد اطمینان می‌دهد که بازی همیشه قابل اجرا است، حتی بدون دارایی‌های اضافی.
