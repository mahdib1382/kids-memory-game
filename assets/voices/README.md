# Persian Voice Files for Numbers

این پوشه شامل فایل‌های صوتی اعداد به زبان فارسی است.

## فرمت فایل‌ها

- فرمت: MP3
- نام‌گذاری: `{number}.mp3`
- مثال: `1.mp3`, `2.mp3`, `10.mp3`, `20.mp3`

## محدوده اعداد

فایل‌های صوتی برای اعداد 1 تا 20 مورد نیاز است.

## ویژگی‌های صدا

- **زبان**: فارسی
- **جنسیت**: مرد (ترجیحاً)
- **سبک**: واضح و آهسته برای کودکان
- **مدت زمان**: کوتاه (< 2 ثانیه)
- **کیفیت**: کیفیت بالا برای وضوح

## مثال محتوا

- `1.mp3` - "یک"
- `2.mp3` - "دو"
- `3.mp3` - "سه"
- `10.mp3` - "ده"
- `11.mp3` - "یازده"
- `20.mp3` - "بیست"

## استفاده در بازی

هنگام باز شدن هر کارت، عدد روی کارت به صورت صوتی خوانده می‌شود تا:
- کودکانی که در خواندن اعداد مشکل دارند یاد بگیرند
- یادگیری اعداد دوچندان سریع‌تر شود
- بازی تعاملی‌تر و آموزشی‌تر باشد

## تهیه فایل‌های صوتی

می‌توانید از روش‌های زیر استفاده کنید:
1. ضبط با گوینده حرفه‌ای
2. استفاده از سرویس‌های text-to-speech فارسی
3. استفاده از کتابخانه‌های TTS مانند gTTS با پشتیبانی فارسی

## توجه

اگر فایل صوتی برای عددی موجود نباشد، بازی بدون گوینده برای آن عدد ادامه می‌یابد.
